回不去的江南景

2020-03-12

高中时碰到鲁迅的文章，一般都非常喜欢。因为表达就是江浙人的风格，读起来很亲切。可是一到考试就不亲切了，“迅哥儿”短短的一句话会被无尽地解读，回答题目时往往让人大费周折，对于没有太多文学细胞的人来说，这个时候实在是喜欢不起来。

现在因为疫情赋闲在家，没有什么时期读读书，翻到了鲁迅的《呐喊》，想起小时候被老师要求背鲁迅的文集：《呐喊》《彷徨》《野草》《朝花夕拾》，现在这么大了却一本也没有完整地读过，不免觉得自己有种错过了国粹的感觉。拈来一读，发现很多文章在原来的课本中已经学过。

印象最深刻的还是下面这段话：

两岸的豆麦和河底的水草所发散出来的清香，夹杂在水气中扑面的吹来；月色便朦胧在这水气里。浅黑的起伏的连山，仿佛是踊跃的兽脊似的，都远远的向船尾跑去。

这里用了很多拟人（不知不觉还是搬来了初高中阅读分析的手法），让画面在我脑海中呈现。江南夜色中的田野，被植物的气息所填满。船两边水墨化般的画面，飞也似地向后飘去，如一滴墨水坠入一碗水中，散开，接着褪去。

这是怎样一个回不去的年代，这是怎样一个叫人羡慕的年龄。历史的大车轮滚滚向前，那些小时候经历的事，都只能留在脑海里了。

2020-02-23

通篇读下来，作者大致想强调的是不会变的，毕竟那是二十多年生活得出的“璀璨钻石”：

  1. 不要被自己的情绪所绑架，自己内心要足够富足强大
  2. 机会不是给有准备的人的，而是给追着机会跑的人
  3. 细水长流，厚积薄发，天道酬勤
  4. (…)

我能总结的差不多以上三点，知道有遗漏就强行加了第四点。
佩服Nicol的内心强大，在考研路上有那么多的人给予负能量，能披荆斩棘扛过来。相比之下，我的爸妈好太多了，给我很多鼓励。虽然也劝我放弃报考现在就读的学校，但幸亏我坚持下来了，正所谓文中所说“他们不用对你的未来负责”，做决定不要被他人左右，知道自己要什么，我得到的就是我应得的。

去年以来，Nicol的文风跟考研期间有蛮大变化，一来是发文数量少之后长期练习的沉淀，另一方面我也感受到金钱这个躲不开的“洪水猛兽”对每一个人的影响。

总之，通篇读来给人内心的冲击比较大，真的适合身心疲惫时打打鸡血。

通过利率来分析银行之间的差异

2020-02-12

判断大-中小银行之间的利差，可以从农商行与国有银行间同业存单利差来判断。比如存单利差从35BP下降到13BP，AA-与AAA同业存单利差从70BP下降至45BP，说明原先市场偏好各规模银行之间的流动性分层问题有所释缓。适度的同业利差可以充分反映不同类型银行的信用风险。而包商银行冲击、打破同业刚兑增加了低评级中小银行同业存单发行难度。

观察R007与DR007的利差是否处于历史高位，可以观察银行与非银层面的流动性。

而查看民营企业与地方国企间的信用利差，可以知道民营企业和国有企业的融资难度差异

浅谈中国总供求分析

2020-02-10

去年（2019年）有这么一种说法：中国总需求不足，导致中国经济增速下行。怎么判断中国总需求不足？消费吗？可以的。

下文将介绍高善文老师在《经济运行的逻辑》之“观察产能周期”所介绍的，基于国际收支平衡的总供求分析框架。

相比传统的Y = C + I + X 支出法分析，它克服了支出法要求总供求稳定的假设。但它也存在缺陷，对可贸易部门和不可贸易部门两者的相互作用进行了简化和省略。

本文将按照以下框架进行介绍：

  • 分析框架简介
  • 中国的资本约束

分析框架简介

基本假设：

（1）经济体分为可贸易部门和不可贸易部门；（2）汇率大体固定；（3）经常账户开放，资本账户封闭；（4）总量波动不影响全球经济平衡格局（小国经济假设）

（小国经济假设解释：大国经济假设的分析过程相当于封闭经济模型，所以在开放经济下不适用。现实往往处于两种极端之间，且不可贸易部门的分析必然是封闭经济模型）

可贸易部门和不可贸易部门

可贸易部门和不可贸易部门的区别在于，可贸易部门的不足或过剩可以通过贸易手段来解决。分别为两者建立起总供给（AS）和总需求（AD）的模型。

不可贸易部门的价格和产量，由AS和AD曲线的交点决定。可贸易部门的价格由外生决定；例如，当价格高于均衡价格时，产出大于需求，就会形成经常账户盈余（CAS）。

将可贸易部门和不可贸易部门的AS曲线和AD曲线合并起来，需求曲线对应国内消费和投资，供给曲线对应总产出。

曲线的移动

世界经济景气程度的改善、预期的改善或政府的政策刺激都会使总需求曲线向右平移；

一个国家的技术进步加速、有效资本存量扩大或劳动者就业倾向改善等冲击都会使总供给曲线向右平移；

如果全球需求扩张，会导致可贸易品价格上升。因为可贸易部门和不可贸易部门在现实中会相互影响，所以不可贸易部门的供给曲线会向左移动，价格上升产出下降。
如果贸易条件改善，我们可以理解为贸易冲击，投入品价格下降，可贸易部门和不可贸易部门的供应曲线同时向右侧移动。

（在更现实的条件下，国际的需求改善会刺激投资活动，需求曲线也会向右移动）

冲击来源识别

主要关注的指标是：可贸易部门和不可贸易部门的产出和价格。

面对国内供求冲击时，两个部门的变化方向是一致的

国际需求冲击时，可以通过观察可贸易品价格变动和国际产出变化来识别和确认。

总结

针对三种冲击来源（国内需求、国内供给、全球供求），我们有以下指标来识别和辨认：经常账户盈余、总产出、可贸易品价格、不可贸易品价格以及全球经济产出（用OECD工业产出指数来代表）

中国的资本约束

与发达经济体相比，推动中国这种追赶型经济体经济增长的主要因素，是资本的快速形成和积累，技术进步扮演了次要的影响，而劳动力供应对中国中长期经济增长的影响并不显著。

发达国家人均资本的占有数量显著高于发展中国家，在历史上，当中国经济的扩张周期运行到接近顶点时，资本存量的约束就会鲜明地表现出来。例如，2003年中国经济处于扩张导致的严重过热阶段，此时煤电油运季度紧张，产能利用率达到顶峰，企业盈利水平处于峰值，不可贸易部门的价格压力非常强烈（因为不可贸易部门的价格往往有政府管控）

如果资本存量有效扩大，那么劳动力市场就会出现就业扩大与工资加速上升并存的局面。

i浅谈通货膨胀的分析方法

2020-02-05

本文是我对高善文老师《经济运行的逻辑》之“分析通货膨胀”的读后感，后续还会有“产能周期”和“资产重估”。

题外话：经济分析的方法论

做宏观经济分析，在对过去几年经济数据全面把握的基础上，要重点回答三个问题：经济走向；政策立场；资金松紧

数值的大小和误差倒是其次，重点是关注方向和趋势。例如政策立场偏宽松，或者相当宽松，这是重点；至于加息次数、贷款规模等，相对次要一些。

通货膨胀分析的框架

本文的宏观分析主要分为以下几部分：在我国，可贸易部门占了经济的绝大部分。所以，我们先从这一部分入手，分析可贸易部门的通货膨胀。此外，我国CPI中食品占了很大的比重，所以要对食品类通货膨胀进行分析。其次，很多人认为货币超发会引发通货膨胀。国外也有人使用菲利普斯曲线预测通货膨胀。根据以上几个问题，我简单将通货膨胀进行梳理。主要框架如下：

  • 可贸易部门的通货膨胀
  • 食品类通货膨胀
  • 货币供应与通货膨胀
  • 菲利普斯曲线与中国通货膨胀预测

可贸易部门的通货膨胀

我国的可贸易部门在中国整体的经济中，占了相当大的比重。从上中游的工业品，如原油、铁矿、钢材等，到下游的电脑、汽车等都是可贸易品。

根据一价定律，在自由贸易、资本开放、无交易成本的情况下，可贸易品全球统一定价，各国之间没有价差。所以，中国的PPI（1996年以后，因为1996年中国放开经常项目的货币兑换）、美国的PPI与欧洲的PPI的波动、拐点应当有较大的一致性。而事实确实如此，中国可贸易品价格波动与全球工业品价格的波动相当一致。

是哪些因素主导了PPI的波动呢？从经验数据来看：美元汇率的波动，大宗商品市场的冲击以及全球总供应层面的变化对PPI产生了重大影响。

全球总供应层面的变化

我们使用OECD工业生产指数来衡量全球工业需求，从过去20年的情况来看，OECD工业生产指数领先美国PPI大约三个月左右。而OECD工业生产指数的波动与美国PPI基本一致，这说明全球工业需求能够影响世界工业品的价格。

我们也可以使用“中国工业增加值同比”和“实际进口同比”来说明中国工业需求的波动直接影响进口需求，进而影响世界工业品的价格。

综上，全球总供给层面的变化都会影响资产价格。

大宗商品市场的冲击

大宗商品市场的价格主要关注地缘政治和战争即可。因为原油等价格变化会直接影响下游产品的成本，所以突发性的黑天鹅事件对可贸易品价格具有重要的影响。

美元汇率的波动

从理论上来说，本国汇率与PPI呈反向关系。因为本国汇率升值，意味着全球可贸易品价格的下降，这会压低本国生产者的价格。如果我们比较“美元指数同比变化”与“美国PPI较全球变化”，我们可以发现，在一部分时候两者呈反向关系，与理论相符。但有时候，美元汇率与PPI呈现同向关系。这是因为美国相对于其他经济体而言，出现周期性走弱，那么美元汇率贬值，美国PPI相对于其他经济体也会走弱。

因为中国的汇率在一定程度上是钉住美元的，所以美元的汇率的变化会对我国通货膨胀产生影响。

食品类通货膨胀

中国的CPI的波动主要由食品价格所主导，且中国食品价格的波动与工业部门的通货膨胀高度同步（我们可以用CPI食品与第二、三产业的平减指数的相关系数来评价，但第三产业很多部门采用CPI作为价格指数，所以使用第二产业的平减指数更为恰当）。是什么引起了这个现象？

猜想1：食品供需影响了工业品价格
这种说法不完全是对的。因为生产资料的价格具有全球同步性，如果因为中国食品价格的上升，引起了中国工业品价格的上升，进而造成全球的工业品价格上升？这种逻辑犯了因果倒置的错误。

猜想2：某些因素同时引起了食品和工业品价格的波动。
中国的经济体系复杂，难以仅用某些因素来解释。

猜想3：整体经济诱发食品通胀。
除了食品供需层面外，存在两种机制将农产品与非农产品联系起来：成本传递与通货膨胀预期。

1.成本传递：化肥、农药、汽油等价格的上升，推高了农户的生活生产成本，通过成本加成的渠道影响粮食价格。
我们通过比较“农产品生产价格指数”与“农产品生产资料价格指数”，我们发现农产品生产资料价格上升的幅度小于农产品生产价格指数上升幅度。如果成本传递说法正确，那么农产品生产资料价格上升的幅度应该大于农产品生产价格指数。所以成本传递并不能解释整体经济对食品价格的影响。

2.通货膨胀预期：农户观察到粮食、农作物价格即将上涨，产生了通货膨胀的预期。这时，农户会减少现金持有量，增加农产品持有量，如延长生猪出栏时间等。

我认为，更重要的是，通货膨胀预期与库存调整假说相关。对于一些容易保存的食品，农户对库存的调整会引起农产品上价格的变动，毕竟在各种市场上，农产品市场更加接近完全竞争市场。所以，根据库存调整假说，对于新鲜蔬菜、新鲜瓜果、生鲜等易腐烂食品而言，这些食品的价格对第二产业平减指数的回归系数、相关系数要偏小；即使在通货膨胀的预期下，因为不易保存的特性，使得他们难以进行存货调整。

通过将易保存食品、不易保存食品与第二产业平减指数回归并进行比较，我们会发现结果支持我们的假设。

为什么工业品的价格遵循一价定律，而食品价格由通货膨胀决定？
因为在我国，农产品市场存在贸易壁垒，相对封闭，初级农产品较为标准化，需求的价格刚性较强。（这也是为什么贸易战要打开我国农产品市场的原因）

那么农户又是根据什么形成通货膨胀预期的呢？接下来我们要结合刘易斯拐点来分析

刘易斯拐点：刘易斯将发展中国家经济区分为现代部门和传统部门，现代部门以制造业为代表；传统部门以农业为代表。制造业以利润最大化为导向，存在持续的技术进步，资本存量是其唯一的约束，企业家利用留存收益不断扩张部门。而农业部门生产方式落后，劳动生产率低下，整个部门存在大量剩余劳动力。而这些剩余劳动力为现代部门提供源源不断的劳动力，使得劳动力供给曲线呈水平。

但当农业部门的剩余劳动力被充分吸纳的时候，劳动力在农业上的边际产出开始高于其维持生计的水平，那么制造业部门的劳动供给曲线由水平变成向右上方倾斜。工资水平开始随着就业扩张而快速上涨。

这一个拐点就是所谓的“刘易斯拐点”

中国大约在2005年左右走过刘易斯拐点，这意味着价格层面表现出来的就是食品价格快速上升，但是工业品并没有那么大的压力。

尤其是那些劳动密集型的农产品，劳动成本在食品价格中占比越大，这两年的累计涨幅也就会越大。1990-2000年的劳动力成本并没有快速上涨，所以在那几年，各种农产品的价格涨幅比例没有太大差别。相比之下，2000-2007年的劳动力成本上涨较大。所以大豆这种农作物，相比于可以机械化耕作的小麦，在价格涨幅上更加凶猛。

再通过比较蔬菜价格同比差（农村 - 城市），我们可以知道，造成CPI蔬菜价格中轴上升的原因，是生产成本，而非农村到城市的流通成本。

回到最初的问题，农户是根据什么形成通货膨胀预期的呢？其实，随着工业化进程的推进，农民外出打工的机会成本出现了大幅度的上升。

那么机会成本又通过通货膨胀预期，传导至粮食价格大幅上涨。

总结一下

由于我国农产品市场相对封闭，农户的通货膨胀预期，会进行库存调整，进而引起食品的价格上涨。农户的通货膨胀预期是根据外出打工的机会成本形成的，且易保存的食品受影响更大。

近年来，CPI食品与CPI指数的价格裂口越来越大。这与日本、韩国、中国台湾等国家和地区的历史表现特别相近。在经过刘易斯拐点之后，CPI食品与CPI指数会有较大的差距。

货币供应与长期通货膨胀

曾经在很长一段时间，经常有人说出这种“虎狼之词”：央妈放水啦！又要通胀啦！价格又要飞涨啦！

其实近十几年来，中国人民银行一直在强调，坚决不搞大水漫灌。其实货币政策与通货膨胀的联系正在逐渐变得松散。

这种错误的观点认为受制于内外平衡以及经济转型调整，降息空间很小。该观点混淆了货币政策逆周期调节和货币超发的关系，大水漫灌的确会产生通货膨胀、资产泡沫，延缓过剩产能的出清，但在经济下行期，正常的货币政策逆周期调节能有效托底经济，为改革和转型赢得时间窗口，兼顾稳增长、调结构和促改革的多重目标。（来源：泽平宏观）

我们通过收集数据，包含1990年至2009年的89个国家的通胀率和货币增速，并对其分组：通胀低于5%，通胀介于5%~20%，通胀高于20%。

通过回归，我们可以证实上述表述，货币政策正在逐渐与通货膨胀脱钩：回归系数随着时间推进变小，拟合优度也在变小。

这种不显著关系，通常存在以下几种解释：

  • 通胀度量的不完整

中国的CPI只统计了人们的日常生活用品的价格。但并不包含住宅建筑的价格。

  • 货币的度量不准确

由于金融创新，货币的外延越来越模糊

  • 央行的公信力取代了货币供应量，成为稳定通胀的“名义锚”

有些国家的中央银行不具备这种公信力，他们通常采用固定汇率政策，于是固定汇率政策成为“名义锚”，而非该国的货币供应量。

所以，在判断长期通货膨胀时，不能仅仅以货币增速的高低作为判断的基础

菲利普斯曲线与中国通货膨胀预测

1.中国的可贸易部门在经济中具有非常大的比重，所以其波动对消费物价指数的影响不可忽略，所以可以视为开放的经济体。成熟经济虽然高度开放，但制造业占比很小，所以可以视为封闭的经济体。而菲利普斯曲线的假设前提就是封闭的经济体。

2.主导中国消费物价指数走向的食品类价格，驱动农户通货膨胀预期的生产资料价格等不存在明显的价格粘性，其波动可以迅速反映市场供求变化的影响。
而在国外成熟经济体中，在用工方面存在“菜单成本”、劳动力价格存在较大的价格粘性。这也是为什么，经过新古典凯恩斯主义修正后的菲利普斯曲线在国外适用性较好的原因。

3.在要素禀赋方面，我国与成熟经济体存在很大不同：我国资本存量制约是制约中国经济增长的主要因素，而成熟经济体中是劳动力短缺。中国生产资料价格变化的背后主要反映了企业生产能力的余缺和资本存量的多寡。

因此，在利用失业率预测未来通货膨胀方面，成熟经济体的菲利普斯曲线对于我国可能并不适用。

以上就是对中国通货膨胀分析框架的简单阐述。
