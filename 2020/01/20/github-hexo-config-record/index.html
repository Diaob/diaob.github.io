写这篇教程，主要想说明一个道理：“关关难过关关过”，给自己打打气，也表明写博客（开坑）的决心。

Q：为什么选择在Github上搭建博客？
A：因为我想要沉浸式的写作体验，国内的博客的功能普遍过于冗杂，朴素的Vim界面才更加适合我（当然也欢迎Emacs党的朋友）

下面简述配置过程，并且给出我碰到问题的解决方法。每个人的问题肯定是不同的，这时候要充分利用搜索引擎，勤动手，多思考。

我的初始环境：Windows10 + Ubuntu（WSL)
