疯狂吸收，野蛮生长，能量守恒

2020-02-23

本来这不应该算一本书的，但Nicol学姐的文笔确实好，她的文风给人一种“老子不怕你”的霸气，勇敢而不失幽默，自我调侃中散发着自信。粗略看一遍，真的很给人正能量。因为我是倒叙看的，透过这几年（2016~2020）的微博，看到她思想的逐步转变，在有些观点上甚至有“打脸”的情况出现。

但是没有人能够完全守住初心，始终如一以不变应万变的本质是“逆水行舟，不进则退”，思想上的转变正是社会洪流对每个人吞噬下的产物。微博合集也可以看作是从考研征途一路走来，到踏入社会施展宏图的心路历程，激昂中略带点丧气恰是当代年轻人的画风。其实仔细通读全部的微博，可以发现一些线索，根据这些线索顺藤摸瓜找到当年研招网录取名单，可以看到学姐名字叫“江乐”，名字如其人吧，丧是一时的心境，但打起鸡血来，最后都会变成一个斗志满满的人。

通篇读下来，作者大致想强调的是不会变的，毕竟那是二十多年生活得出的“璀璨钻石”：

  1. 不要被自己的情绪所绑架，自己内心要足够富足强大
  2. 机会不是给有准备的人的，而是给追着机会跑的人
  3. 细水长流，厚积薄发，天道酬勤
  4. (…)

我能总结的差不多以上三点，知道有遗漏就强行加了第四点。
佩服Nicol的内心强大，在考研路上有那么多的人给予负能量，能披荆斩棘扛过来。相比之下，我的爸妈好太多了，给我很多鼓励。虽然也劝我放弃报考现在就读的学校，但幸亏我坚持下来了，正所谓文中所说“他们不用对你的未来负责”，做决定不要被他人左右，知道自己要什么，我得到的就是我应得的。

去年以来，Nicol的文风跟考研期间有蛮大变化，一来是发文数量少之后长期练习的沉淀，另一方面我也感受到金钱这个躲不开的“洪水猛兽”对每一个人的影响。

总之，通篇读来给人内心的冲击比较大，真的适合身心疲惫时打打鸡血。

另外，我用Python通读了Nicol的全部微博，然后提取了关键词做了词云。结果如今通读完才发现，以我的编程水平，Python还是无法代替耐心阅读的啊~
